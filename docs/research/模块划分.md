| **模块名称 (团队)**                      | **负责人画像**            | **核心研究内容与作用**                                       | **需要的“输入数据” (来源)**                                  | **最终输出的“产物” (交付物)**                                |
| ---------------------------------------- | ------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 1**.BioDriver（生物驱动/发现层）**       | 计算生物学家 / 生信工程师 | **致病机制挖掘与功能注释：** 利用 AlphaGenome/Phosformer 进行多模态组学分析；**集成 PTM 验证子模块**，评估蛋白磷酸化/甲基化位点的活性改变。 | **华大/公用库：** WGS/WES、单细胞矩阵、DEGs、蛋白 Fasta 序列。 | **提供给下游**：① 经 PTM 风险评估的致病靶点。② 疾病特征签名 (Signature) |
| **2. DrugReflector (表型药物研发)**      | 化学信息学工程师          | **老药新用：** 拿着疾病特征，在几万种已有药物库中匹配，寻找能“逆转”疾病状态的小分子药物。 | **中台提供：** 疾病特征签名 (Disease Signature)。            | **候选小分子药物清单**（准备拿去做细胞毒性实验）。           |
| **3. RNAi-Therapy (RNA 干扰疗法)**       | 核酸药物工程师            | **基因静默：** 针对中台发现的致病基因，设计 siRNA 序列，从源头阻止坏蛋白的生成。 | **中台提供：** 致病靶点（具体的基因序列信息）。              | **siRNA 候选序列**（准备做细胞敲低验证）。                   |
| **4. PrecisionDelivery (ADC 生物导弹)**  | 蛋白质工程专家            | **精准打击：** 针对中台发现的细胞表面突变蛋白，设计特异性抗体，并连接化疗药物。 | **中台提供：** 致病靶点（具体的细胞膜表面蛋白质 3D 结构/类型）。 | **ADC 候选大分子**（抗体-药物偶联物设计图）。                |
| **5. ImmunoGen (mRNA 疫苗)**             | 免疫学 / AI 序列工程师    | **激活免疫：** 利用病人的特异性突变（新抗原），设计 mRNA 序列，教导病人自己的免疫系统去杀癌细胞。 | **中台提供：** 致病靶点（肿瘤特有的突变多肽序列 / Neoantigen）。 | **mRNA 疫苗序列**（准备包裹进 LNP 脂质体进行测试）。         |
| **6.Simulation Hub** (物理仿真/验证中台) | 计算化学家 / MD 模拟专家  | **高精度物理化学验证：** 构建显式溶剂系统，利用 **OpenMM** 运行长程分子动力学（MD）；执行 **AM1-BCC** 电荷校正与自由能计算。 | **Upstream (Bio/Drug 模块)：** 靶点 PDB 结构、小分子 SDF/Mol2、力场参数。 | ① 结合自由能 (delta G) 报告                                  |

BioDriver（生物驱动模块）、PrecisionDelivery (精密递送/ADC)、RNAi-Therapy (RNA 干扰模块)、ImmunoGen (免疫基因模块)、DrugReflector（药物表征模块）、Simulation Hub** (物理仿真/验证中台)

**【L0 / L1 级基础数据】—— 由 BGI（华大）提供**

- **基因表达矩阵 (Raw Count Matrix)：** 请提供 `.h5ad` 或 `.rds` 格式（包含所有细胞和基因）。
- **细胞注释文件 (Cell Type Annotation / Metadata)：** 包含每个 Barcode 对应的细胞分群信息（UMAP/tSNE 坐标及鉴定出的细胞类型）。
- **差异表达分析表 (DEGs)：** 各细胞群（特别是肿瘤细胞 vs 正常细胞）的差异基因表格，需包含 log2FC 和 p-value。”

- HLA 分型数据 (HLA Typing)（
    - 你们可以利用开源算法（如 `OptiType`、`HLA-HD` 或 `arcasHLA`）直接从血液的 WES 原始数据中“算”出 HLA 型号。是这样吗）
- scRNA-seq数据要吗？
- WES（**ImmunoGen**（疫苗）和 **PrecisionDelivery**（ADC）、RNAi-Therapy  足够了吗还是说都用WGS效果更好？）

【L2 级高价值情报】—— 由 BioDriver 提取和医院确认，提供给下游

- **致病靶点 (Pathogenic Target)：** 明确指出具体的突变基因/蛋白（如：发现 *KRAS G12C* 突变且高度磷酸化激活）。
- **序列靶点（突变多肽） / 新抗原(Neoantigen)：**专门给mRNA 疫苗
- **疾病特征签名 (Disease Signature)：** 经过 AI 去噪、临床文献核对后，提炼出的最能代表该癌症的 200 个核心差异基因列表（带有折叠变化率 LFC）。
